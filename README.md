# 笛の熱量 — Road to Drone hackathon

2026 FIFAワールドカップ南米予選のファウル・カード・出場停止データを巡る、全12シーンの3Dストーリーマップです。

## 公開

GitHub Pages: **[https://furuhashilab.github.io/2026-/](https://furuhashilab.github.io/Drone-hachathon/#scene-1)**

依存するビルド工程はありません。静的ファイルをリポジトリ直下から配信します。

## 構成

- MapLibre GL JS 5.24.0
- OpenFreeMap / OpenStreetMap ベクトルタイル
- Esri World Imagery（国境確認時のみ）
- Natural Earth 1:110m 国境データ（ローカル同梱）
- 12カメラシーン。各国の直前だけ俯角0度の国土確認を挟み、都市・スタジアムへ段階的に傾けて接近
- 自動カメラ移動、自動ポップアップ、自動ループ
- 主要予選会場を舞台にした簡略3Dスタジアム、3D建物、ファウル数に限定した統計柱
- キーボード操作：左右キーで移動、Spaceで再生・一時停止

## データ定義

- 総ファウル：18試合 × 公開されている1試合平均（四捨五入した推定値）
- イエロー／レッド：AiScoreの大会別チーム統計
- 出場停止者：予選中に少なくとも一度処分を消化したユニーク選手数
- Bolivia / Paraguay：統計サイトとDiscipline表に不一致があるため信頼度C

## スタジアムの扱い

各代表のシーンは、ホーム9試合で最も使用回数が多かった会場へ移動します。ブラジルはマラカナとマネ・ガリンシャが各2試合で同率のため、マラカナを採用しています。

スタジアムはOpenStreetMapで確認した実位置に配置していますが、形状は表示の安定性と軽量化を優先した簡略3Dモデルです。表示されるファウル・カードは会場別ではなく、従来どおり代表の予選全18試合の統計です。

国別シーンは推定総ファウルの少ない順（Argentina → Chile → Bolivia → Brazil → Ecuador → Peru → Uruguay → Venezuela → Paraguay → Colombia）に進みます。フォーカス時の3D柱とカード内グラフはファウル数だけを表示し、イエロー、レッド、出場停止、会場使用数は自動ポップアップにまとめています。

各国への遷移は、国土を見下ろすトップビューと国旗・国名のタイトル表示から始まり、都市への斜め移動、スタジアムへの最終進入という3段階で構成しています。トップビュー以外のシーンでは従来どおり斜めカメラを維持します。

情報は一度に表示せず、約15秒の中で「航空写真と発光する国境 → 国旗・国名 → ファウル柱と最小限の3数値 → イエロー・レッド・出場停止の3カード → 数値背景」の順に段階表示します。補足カードは画面右上へ固定し、スタジアムと左側グラフを避けます。

出典リンクと解釈上の注意は、画面右上の `i` から確認できます。
